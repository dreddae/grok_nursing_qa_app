# Grok Nursing QA App

This Streamlit app provides a Q&A interface for nursing-related queries.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/dreddae/grok_nursing_qa_app.git
   cd grok_nursing_qa_app
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   streamlit run app.py
   ```
